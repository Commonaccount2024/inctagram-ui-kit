# Usage:

- Clone template: (replace `my-lib` with your project name)
  ```bash
  npx degit https://github.com/ershisan99/lib-with-storybook-starter my-lib
  ```

- Change directory: (replace `my-lib` with your project name)
  ```bash
  cd my-lib
  ```

- Install dependencies:
  ```bash
  pnpm i
  ```

- Run storybook:
  ```bash
  pnpm dev
  ```
